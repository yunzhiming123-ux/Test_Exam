# 更新日志

## v2.0 (2026-05-23)

### 新增
- 练习模式 & 考试模式（原只有学习模式）
- 选择题交互：点击选项 → 提交 → 自动判对错
- 填空题交互：输入框填写 → 提交 → 逐空比对
- 顶部计分栏（练习模式） / 计时栏（考试模式）
- 搜索框（学习模式下过滤题目）
- 键盘快捷键 `N` 跳下一题
- CSS 动画：卡片淡入、对错反馈弹跳、按钮涟漪

### 修复
- **Bug**：同行多个选项（`A. xxx  B. yyy`）只能识别第一个 → 已修复，现在正确拆分为 4 个独立选项
- **Bug**：选择题答案比对取错字母（从 `**A4.** B` 开头取 `A` 而非 `B`） → 已修复，智能提取正确选项字母
- **Bug**：非选择题的答案也误匹配了字母 → 已修复，仅选择题使用 `answer_letter`

### 变更
- 文件 08 补充了 72 道题的 `<details>` 答案块（从底部答案区提取）
