# MD 互动练习器

> 将 Markdown 题库变成可交互的练习工具  
> 支持三种模式 × 三种主题，自动识别选择题、填空题、简答题、代码分析题

## 快速开始

### 运行（Python）

```bash
# 原版阅读器
python MyFirstQt.py

# 互动练习器（推荐）
python md_practice.py
```

### 打包为 exe

```bash
# Windows
双击 build_exe.bat
# 输出: dist/MD练习器/MD练习器.exe

# Linux
bash build_linux.sh
# 输出: dist/MD练习器/MD练习器
```

## 功能说明

### MyFirstQt.py —— 优雅 Markdown 阅读器

| 功能 | 说明 |
|------|------|
| 主题切换 | 亮色 / 暗色 / 护眼 三套预设主题 |
| 自定义颜色 | 菜单 → 主题 → 自定义背景/文字颜色 |
| 工具栏 | 顶栏下拉框快速切换主题 |
| MathJax | 数学公式渲染（CDN） |
| 代码高亮 | highlight.js 语法高亮（CDN） |

### md_practice.py —— 互动练习器

| 模式 | 说明 |
|------|------|
| **学习** | 完整阅读内容，点击"显示答案"折叠查看答案 |
| **练习** | 逐题作答（选择/填空），提交后自动判对错，顶部实时计分 |
| **考试** | 统一计时，全部答完后提交，百分制评分 |

| 主题 | 说明 |
|------|------|
| **亮色** | 白色卡片 + 蓝色强调色，适合白天 |
| **暗色** | 深色背景 + 柔和配色，适合夜间 |
| **护眼** | 米黄底色 + 暖色调，长时间不疲劳 |

| 题型支持 | 交互方式 |
|----------|----------|
| 选择题 | 点击选项 → 提交 → 自动判对错 + 高亮正确选项 |
| 填空题 | 输入框填写 → 提交 → 逐空比对答案 |
| 简答题 | 折叠显示参考答案，自行比对 |
| 代码分析题 | 显示代码块 + 折叠答案 |
| 计算题 | 折叠显示参考答案 |

| 其他功能 | 说明 |
|----------|------|
| 搜索框 | 学习模式下可搜索题目关键字 |
| 快捷键 | 按 `N` 跳转到下一题 |
| 主题切换 | 左下角下拉框选择亮色/暗色/护眼 |
| 考试打乱 | 考试模式下选项顺序随机化 |

## 题库语法（编写自己的 .md 文件）

详见 `模板_题库格式示例.md`。核心规则：

1. 每道题的答案放在 `<details><summary>答案</summary>...</details>` 中
2. 选择题选项用 `A. xxx` `B. xxx` 每行一个（或同行的 `A. xxx  B. xxx` 也支持）
3. 填空题横线用 `___` 或 `\underline{}`
4. 题型可选标记：`（选择题）` `（填空题）` `（简答题）` `（代码分析题）`
