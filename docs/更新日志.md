# 更新日志

## v2.2 (2026-05-23)

### 修复
- 主题下拉框选择后不实时刷新 → `_on_theme_change` 调用错误方法名
- 学习模式缺 MathJax/highlight.js CDN → 公式和代码高亮不显示

### 变更
- 学习模式改为直接渲染原始 Markdown（不再解析题目卡片），仅练习/考试模式提取题目
- 更适合复习阅读：不会被复习要点误识别为题目打断阅读流

## v2.1 (2026-05-23)

### 新增
- 主题系统：亮色 / 暗色 / 护眼 三套主题
- 自定义颜色：左侧面板"自定义背景""自定义文字"按钮 → 取色器
- 重置按钮：恢复当前主题默认颜色
- 全主题 CSS 动画：卡片淡入、选项涟漪、对勾弹跳、错误抖动、光晕脉冲

### 修复
- **Bug**：题目序号重复导致答案按钮失灵 → QID 改为全局唯一计数器
- **Bug**：section 间 Q1 重复导致 DOM id 冲突 → 每题分配唯一内部ID，保留原始题号显示

## v2.0 (2026-05-23)

### 新增
- 练习模式 & 考试模式（原只有学习模式）
- 选择题交互：点击选项 → 提交 → 自动判对错
- 填空题交互：输入框填写 → 提交 → 逐空比对
- 顶部计分栏（练习模式） / 计时栏（考试模式）
- 搜索框（学习模式下过滤题目）
- 键盘快捷键 `N` 跳下一题
- CSS 动画：卡片淡入、对错反馈弹跳、按钮涟漪

### 修复
- **Bug**：同行多个选项（`A. xxx  B. yyy`）只能识别第一个 → 已修复，现在正确拆分为 4 个独立选项
- **Bug**：选择题答案比对取错字母（从 `**A4.** B` 开头取 `A` 而非 `B`） → 已修复，智能提取正确选项字母
- **Bug**：非选择题的答案也误匹配了字母 → 已修复，仅选择题使用 `answer_letter`

### 变更
- 文件 08 补充了 72 道题的 `<details>` 答案块（从底部答案区提取）

## v1.0 (2026-05-21)

- 初始版本
- 12 份 .md 题库文件，总计 302 道题
- Markdown 阅读器（学习模式 + 折叠答案）
- 支持 MathJax 公式渲染 + highlight.js 代码高亮
