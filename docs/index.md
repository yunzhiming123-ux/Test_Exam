# 深度学习考试题库

---

## 📚 项目简介

这是一个**全中文互动学习工具**，专为深度学习学习者设计，支持三种学习模式：

| 模式 | 功能描述 |
|------|----------|
| 📖 **学习模式** | 阅读内容、折叠答案 |
| ✍️ **练习模式** | 逐题作答、自动判对错 |
| 📝 **考试模式** | 限时作答、统一评分 |

---

## ✨ 功能特点

- **自动识别题型**：选择题、填空题、简答题、代码分析题
- **多主题支持**：亮色、暗色、护眼三种主题
- **实时反馈**：答题后立即显示正确答案
- **动画效果**：流畅的交互体验和动画反馈

---

## 🗂️ 题库内容

- **神经网络基础**：线性回归与 DNN
- **卷积神经网络**：CNN 原理与应用
- **循环神经网络**：RNN、LSTM
- **Transformer**：自注意力机制
- **代码分析**：实战代码解读

---

## 🚀 快速开始

### 安装依赖

```bash
uv init
uv add pyqt5 pyqtwebengine markdown
```

### 运行应用

```bash
uv run python exam_bank.py
```
