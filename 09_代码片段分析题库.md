```python
import numpy as np

def cross_entropy_loss(y_true, y_pred):
    """
    手动实现交叉熵损失
    y_true: (N,) int labels in [0, C-1]
    y_pred: (N, C) model logits
    """
    N = y_true.shape[0]
    # Step 1: Softmax
    exp_pred = np.exp(y_pred - np.max(y_pred, axis=1, keepdims=True))
    softmax_pred = exp_pred / np.sum(exp_pred, axis=1, keepdims=True)
    # Step 2: Cross-entropy
    loss = -np.log(softmax_pred[np.arange(N), y_true])
    return np.mean(loss)
```

## 参考答案

### 第一组
**1-1:** (1) DNN/MLP全连接网络 (2) 2层隐藏层 (3) 输入784→输出10 (4) 展平图像为向量 (5) 训练时随机失活，测试时关闭 (6) 分类任务（如MNIST手写数字）  
**1-2:** (1) CNN (2) 32×32 (3) 2层卷积，1层全连接 (4) kernel=3, padding=1保持尺寸 (5) 32→16→8 (6) 64通道×8×8空间 (7) 加速收敛+正则化，放在卷积后、激活前  
**1-3:** (1) LSTM语言模型 (2) 2层LSTM堆叠 (3) 输入shape为 (batch,seq,feature); 否则 (seq,batch,feature) (4) (vocab_size, embed_dim) (5) 文本生成/语言建模 (6) 12个矩阵：4×(W,U)×128×128 + 4×(W)×128×128 = 4×128×128×2 + 4×128×128 = 131072+65536=196608（约）  
**1-4:** (1) Transformer Encoder Block (2) Self-Attention，Q=K=V=x (3) 256/8=32 (4) 1024 (5) Pre-LN 残差连接 (6) 在FFN中扩展维度增加表达能力，再压缩回原维度  
**1-5:** (1) ResNet (2) 快速下采样(4×)减少后续计算，大kernel捕获更多信息 (3) 只在每个stage的第一个block做下采样(改变尺寸和通道数) (4) 替代Flatten+固定FC，可接受任意输入尺寸 (5) layer1: 64→64, 2blocks; layer2: 64→128, 2blocks

### 第二组
**2-1:** (1) conv: (B,64,32,32); pool: (B,64,16,16); fc: (B,10) (2) 3×3×3×64=1728 (3) 64通道×16×16=16384; 但代码写的是64*16*16也是16384  
**2-2:** (1) conv1: (32,64,224,224); pool1: (32,64,112,112); conv2: (32,128,112,112); pool2: (32,128,56,56); conv3: (32,256,56,56); pool3: (32,256,28,28) (2) 224→112→56→28 (3) 256×28×28=200704  
**2-3:** (1) conv保持尺寸50×50，pool后为25×25，32通道 (2) 使用 `nn.AdaptiveAvgPool2d` 替代硬编码尺寸

### 第三组
**3-1:** (1) 问题①没有激活函数，多层卷积退化为线性；问题②32*28*28与实际不符(32→30→28，28×28×32=25088，但32×28×28=25088也是对的...实际上应该仔细检查)；问题③缺少正则化 (2) 添加ReLU，添加BatchNorm，检查维度计算  
**3-2:** (1) 问题①隐藏层用Sigmoid导致梯度消失；问题②模型输出Softmax+CrossEntropyLoss会导致双重Softmax (2) 双重Softmax→概率分布趋于均匀→模型无法有效学习 (3) 改ReLU，去掉最后的Softmax  
**3-3:** (1) `optimizer.zero_grad()` 应在 `loss.backward()` 之前 (2) 累积N步后再调用 `optimizer.step()` 和 `optimizer.zero_grad()`

### 第四组
**4-1:** (1) Embedding+LSTM(2层×4组参数)+MLP(2个Linear)=5组可学习层(粗略) (2) Dropout, LSTM层间Dropout (3) num_layers=2, batch_size, hidden_dim=128 (4) 只取最后一个时间步的输出（用于自回归生成） (5) LogSoftmax+NLLLoss等价CrossEntropyLoss但更稳定 (6) Embedding: 5000×100=500K; LSTM: (100×128+128×128)×4×2×2≈? ; MLP: 128×512+512+512×5000+5000

### 第五组
**5-1:** (1) 防止指数溢出（上溢为inf） (2) 取出每个样本真实标签对应的预测概率 (3) 指数上溢为inf，产生NaN (4) `F.cross_entropy(logits, y_true)`
