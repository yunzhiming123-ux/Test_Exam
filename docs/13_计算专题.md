## 二、Softmax 概率计算

三分类任务 logits：z₁ = 3, z₂ = 1, z₃ = 0

**公式：** $$y_i = \frac{e^{z_i}}{\sum_{j} e^{z_j}}$$

<quiz>
$e^z₁$ = [[20.09]]，$e^z₂$= [[2.72]]，$e^z₃$ = [[1.00]]

$\sum e^{z_j}$ = [[23.81]]

y₁ = [[0.84]]，y₂ = [[0.11]]，y₃ = [[0.04]]

---

| 项 | 计算 | 结果 |
|----|------|------|
| $e^{z_1}$ | $e^3$ | 20.09 |
| $e^{z_2}$ | $e^1$ | 2.72 |
| $e^{z_3}$ | $e^0 = 1$ | 1.00 |
| $\sum e^{z_j}$ | 20.09 + 2.72 + 1.00 | 23.81 |
| y₁ | 20.09 / 23.81 | 0.84 |
| y₂ | 2.72 / 23.81 | 0.11 |
| y₃ | 1.00 / 23.81 | 0.04 |

验证：0.84 + 0.11 + 0.04 = 0.99 ≈ 1（四舍五入误差）
</quiz>

## 三、前向传播 + 误差计算

前馈神经网络，ReLU 激活函数。输入 i₁=0.5, i₂=-0.3，目标 t₁=0, t₂=1。

权重：w₁=0.2, w₂=-0.1, w₃=0.4, w₄=0.3, w₅=0.1, w₆=0.5, w₇=-0.2, w₈=0.6
偏置：b₁=0.1, b₂=0.2

误差公式：$$E = \frac{1}{2}(target - output)^2$$

<quiz>
h₁ = [[0.23]]，h₂ = [[0.21]]

o₁ = [[0.328]]，o₂ = [[0.28]]

E_O₁ = [[0.0538]]，E_O₂ = [[0.2592]]，E_total = [[0.3130]]

---
**隐藏层：**
net_h₁ = 0.2×0.5 + (-0.1)×(-0.3) + 0.1 = 0.23 → h₁ = ReLU(0.23) = 0.23
net_h₂ = 0.4×0.5 + 0.3×(-0.3) + 0.1 = 0.21 → h₂ = ReLU(0.21) = 0.21

**输出层：**
net_o₁ = 0.1×0.23 + 0.5×0.21 + 0.2 = 0.328 → o₁ = ReLU(0.328) = 0.328
net_o₂ = (-0.2)×0.23 + 0.6×0.21 + 0.2 = 0.28 → o₂ = ReLU(0.28) = 0.28

**误差：**
E_O₁ = ½(0-0.328)² = 0.0538
E_O₂ = ½(1-0.28)² = 0.2592
E_total = 0.0538 + 0.2592 = 0.3130
</quiz>

## 四、反向传播权重更新

基于第三题结果，学习率 η = 0.1，求更新后的 w₅。

权重更新公式：
$$w_5^{new} = w_5 - \eta \cdot \frac{\partial E_{total}}{\partial w_5}$$

<quiz>
$\partial E/\partial w_5$ = [[0.0754]]

$w_5^{new}$ = [[0.0925]]

---
**链式法则：**
$\partial E_{O1}/\partial o_1$ = $-(t_1 - o_1)$ = 0.328
$\partial o_1/\partial net_{o_1}$ = 1（ReLU 导数，输入 0.328 > 0）
$\partial net_{o_1}/\partial w_5$ = h₁ = 0.23
$\partial E/\partial w_5$ = 0.328 × 1 × 0.23 = 0.0754

**更新：**
$w_5^{new}$ = 0.1 - 0.1×0.0754 = 0.1 - 0.00754 = 0.0925
</quiz>

## 六、特征图尺寸计算

输入 32×32×3，`Conv2d(3, 64, kernel_size=3, padding=1, stride=1)` + `MaxPool2d(2, 2)`

公式：
$$H_{out} = \left\lfloor \frac{H_{in} - K + 2P}{S} \right\rfloor + 1$$
$$\text{Params}_{Conv} = C_{in} \times C_{out} \times K^2 + C_{out}$$

<quiz>
conv 后输出尺寸：[[32×32×64]]

pool 后输出尺寸：[[16×16×64]]

卷积层参数量：[[1792]]

---
**卷积输出：** $H_{out} = (32-3+2\times1)/1 + 1 = 32 \to 32\times32\times64$
**池化输出：** 32/2 = 16 → 16×16×64
**参数量：** 3×64×3² + 64 = 1728 + 64 = 1792
</quiz>

## 七、链式法则与梯度消失证明

10 层全连接网络，所有隐藏层使用 Sigmoid 激活函数。
σ'(x) = σ(x)(1 - σ(x)) ≤ 0.25

<quiz>
$\sigma'(x)$ 的最大值是[[0.25]]，在 $x$ = [[0]] 时取得

10 层 Sigmoid 网络，梯度最多衰减到：[[9.54×10⁻⁷]]（约百万分之一）

若改用 ReLU，10 层传播后梯度：[[1]]（理论上无损传播）

---
1. $\sigma'(0) = 0.5 \times 0.5 = 0.25$
2. $0.25^{10} \approx 9.54 \times 10^{-7}$ → 梯度消失
3. ReLU 正区间导数为 $1$ → $1^{10} = 1$ → 缓解梯度消失
</quiz>

## 八、交叉熵手动计算

三分类任务，logits = [2.0, 1.0, -1.0]，真实标签 label = 0。

Softmax 公式 + 交叉熵损失：
$$\mathcal{L} = -\sum_{c} y_c \log(\hat{y}_c)$$

<quiz>
$\hat{y}_0$ = [[0.705]]，$\hat{y}_1$ = [[0.260]]，$\hat{y}_2$ = [[0.035]]

Loss = [[0.350]]

---
**Softmax：**
$e^2=7.389$, $e^1=2.718$, $e^{-1}=0.368$, $\Sigma=10.475$
$\hat{y}_0=7.389/10.475=0.705$, $\hat{y}_1=2.718/10.475=0.260$, $\hat{y}_2=0.368/10.475=0.035$

**交叉熵：**
Loss = -log(0.705) = 0.350
</quiz>
