# 功能说明

## 三种模式

### 学习模式
- 直接渲染 Markdown 全文（与 MyFirstQt.py 阅读器相同）
- 保留所有标题、表格、代码块、引用等原始格式
- **不解析题目卡片**，适合复习阅读
- 主题可切换（亮色/暗色/护眼）

### 练习模式
- 每道题独立作答，提交后立即判对错
- **选择题**：点击选项选中 → 点击「提交答案」
- **填空题**：在输入框中填写 → 自动记录 → 点击「提交答案」
- 顶部计分栏实时更新：已答 / 总题数 / 答对
- 正确答案高亮（绿色边框+光晕动画），错误选项划删除线+抖动动画

### 考试模式
- 所有题目统一计时
- 选项顺序随机打乱（防作弊）
- 点击「提交试卷」一次性提交所有答案
- 百分制评分 + 评语

## 主题系统

| 主题 | 特点 |
|------|------|
| 亮色 | 白底蓝调，适合白天 |
| 暗色 | 深蓝底柔色调，夜间不刺眼 |
| 护眼 | 米黄暖色，长时间使用不疲劳 |
| 自定义背景 | 左下角「自定义背景」按钮 → 取色器选色 |
| 自定义文字 | 左下角「自定义文字」按钮 → 取色器选色 |
| 重置 | 恢复当前主题默认色 |

## 题型识别

程序自动从 .md 文件中检测题型：

| 题型 | 判定依据 | 交互控件 |
|------|----------|----------|
| 选择题 | 含 A/B/C/D 选项 或 标记 `（选择题）` | 单选按钮 |
| 填空题 | 含 `___` 或 `\underline{}` | 文本输入框 |
| 简答题 | 无选项标记的普通问答 | 折叠答案 |
| 代码分析题 | 含代码块 | 折叠答案 |
| 计算题 | 标记 `（计算题）` | 折叠答案 |

## 快捷键

| 按键 | 功能 |
|------|------|
| `N` | 跳转到下一题 |

## 动画效果

| 动画 | 触发场景 |
|------|----------|
| 卡片依次淡入 | 页面加载 |
| 选项涟漪 | 点击选项 |
| 对勾弹跳 | 回答正确 |
| 错误抖动 | 回答错误 |
| 按钮悬浮上移 | 鼠标悬停 |
| 卡片悬浮上浮 | 鼠标悬停 |
| 正确答案光晕脉冲 | 显示答案后 |

## 依赖

- Python 3.7+
- PyQt5 + PyQtWebEngine
- markdown
