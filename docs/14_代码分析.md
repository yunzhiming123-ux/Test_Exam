## 第一题：模型结构分析

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class MyClassifier(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(8, 16)
        self.act1 = nn.Sigmoid()
        self.fc2 = nn.Linear(16, 32)
        self.act2 = nn.ReLU()
        self.fc3 = nn.Linear(32, 5)

    def forward(self, x):
        x = self.fc1(x)
        x = self.act1(x)
        x = self.fc2(x)
        x = self.act2(x)
        x = self.fc3(x)
        return x
```

**问题：**

<quiz>
(1) 该网络共有[[3]]层全连接层

(2) 输入特征维度是[[8]]

(3) 输出维度对应[[5]]分类任务

(4) fc1 激活函数：[[Sigmoid]]；fc2 激活函数：[[ReLU]]

(5) 最后一层是否使用了激活函数？[[否]]

(6) 该网络属于[[DNN/MLP]]类型

(7) 参数量最大的层是[[fc2]]，参数量为[[544]]

---

| 空位 | 答案 | 说明 |
|------|------|------|
| (1) | **3 层** | fc1, fc2, fc3 |
| (2) | **8** | Linear(8, 16) 的 in_features=8 |
| (3) | **5 类** | Linear(32, 5) 输出 5 维 |
| (4) | fc1: Sigmoid, fc2: ReLU |
| (5) | **否** | fc3 后直接 return，未加激活 |
| (6) | **DNN/MLP** | 只有全连接层 |
| (7) | **fc2**：$16\times32+32=544$ | fc1: 144, fc3: 165 |

</quiz>

## 第二题：Embedding + LSTM 代码分析

```python
import torch
import torch.nn as nn

class TextModel(nn.Module):
    def __init__(self, vocab_size=20, embed_dim=8, hidden_dim=16):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, embed_dim)
        self.lstm = nn.LSTM(embed_dim, hidden_dim, num_layers=2,
                            batch_first=True)
        self.fc = nn.Linear(hidden_dim, 4)

    def forward(self, x):
        x = self.embedding(x)           # (B, T) → (B, T, 8)
        out, (hn, cn) = self.lstm(x)    # (B, T, 16)
        out = out[:, -1, :]             # 取最后一个时间步
        out = self.fc(out)              # (B, 4)
        return out
```

<quiz>
(1) Embedding(20, 8) 中 20 = [[词汇表大小]]，8 = [[嵌入向量维度]]

(2) Embedding 层参数量 = [[160]]

(3) batch_first=True 时输入 shape 为 [[(batch, seq_len)]]，输出 shape 为 [[(batch, seq_len, embed_dim)]]

(4) num_layers=2 含义：[[2层LSTM纵向堆叠]]；hn shape = [[(2, batch, 16)]]

(5) out[:, -1, :] 取出[[每个序列最后一个时间步的隐状态]]

(6) 该模型适用于[[文本分类]]任务，输出 4 维表示[[4分类]]

---

| 问题 | 答案 |
|------|------|
| (1) | 20=词汇表大小，8=嵌入向量维度 |
| (2) | $20 \times 8 = 160$ |
| (3) | 输入 (batch, seq_len)，输出 (batch, seq_len, embed_dim) |
| (4) | 2 层 LSTM 堆叠；hn shape = (2, batch, 16) |
| (5) | 每个序列最后一个时间步的全局表示 |
| (6) | 文本分类（情感分析等），4 分类 |

</quiz>

## 第四题：错误识别（常见 bug）

找出以下代码片段中的错误并修改。

**代码 A：**
```python
output = F.softmax(model(x), dim=1)
loss = F.cross_entropy(output, y)
```

**代码 B：**
```python
x = x.transpose(1, 2).view(B, -1)
```

**代码 C：**
```python
for x, y in train_loader:
    output = model(x)
    loss = criterion(output, y)
    loss.backward()
    optimizer.step()
```

**代码 D（验证阶段）：**
```python
model.train()
for x, y in val_loader:
    output = model(x)
```

<quiz>
代码 A 错误：[[双重Softmax]]（CrossEntropyLoss 内部已含 LogSoftmax）
修改：output = model(x)，直接传 logits

代码 B 错误：[[transpose后张量不连续，view报错]]
修改：x = x.transpose(1, 2).contiguous().view(B, -1) 或用 reshape

代码 C 错误：[[缺少zero_grad，梯度累加]]
修改：在每个 batch 前加 optimizer.zero_grad()

代码 D 错误：[[验证时用了train模式]]
修改：model.eval() + with torch.no_grad()

---
**代码 A**：双重 Softmax → 移除 forward 中的 Softmax
**代码 B**：transpose 后内存不连续 → 加 .contiguous() 或用 .reshape()
**代码 C**：缺少 zero_grad → 每个 batch 前清空梯度
**代码 D**：训练/验证模式混淆 → eval 模式 + no_grad
</quiz>

## 第五题：完整模型代码分析

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class PoetryModel(nn.Module):
    def __init__(self, vocab_size=5000, embed_dim=100, hidden_dim=128):
        super().__init__()
        self.embed = nn.Embedding(vocab_size, embed_dim)
        self.lstm = nn.LSTM(embed_dim, hidden_dim, num_layers=2,
                            batch_first=True, dropout=0.3)
        self.mlp = nn.Sequential(
            nn.Dropout(0.3),
            nn.Linear(hidden_dim, 512),
            nn.ReLU(),
            nn.Dropout(0.3),
            nn.Linear(512, vocab_size)
        )
        self.log_softmax = nn.LogSoftmax(dim=-1)

    def forward(self, x, is_test=False):
        batch_size = x.size(0)
        embed = self.embed(x)
        h0 = torch.zeros(2, batch_size, 128, device=x.device)
        c0 = torch.zeros(2, batch_size, 128, device=x.device)
        out, _ = self.lstm(embed, (h0, c0))
        if is_test:
            out = out[:, -1, :]
        out = self.mlp(out)
        return self.log_softmax(out)
```

<quiz>
(1) 模型总参数量约[[3364520]]（Embedding+LSTM+MLP两层Linear）

(2) 模型使用了哪些正则化：[[Dropout]]（LSTM层间0.3 + MLP中两层0.3）

(3) h0 第一个维度为 2 的原因：[[num_layers=2，每层都需要初始状态]]

(4) is_test=True 时 out[:, -1, :] 作用：[[只取最后时间步预测下一个字符]]

(5) 输出用 LogSoftmax 原因：[[配合NLLLoss使用，数值更稳定]]

---

| 空位 | 答案 |
|------|------|
| (1) | Embedding: 500K + LSTM: 233K + MLP: 2.63M $\approx$ 3.36M |
| (2) | 3 处 Dropout(0.3) |
| (3) | 2 层 LSTM，每层需要独立的 $h_0$/$c_0$ |
| (4) | 测试/生成时只取最后时间步输出 |
| (5) | LogSoftmax + NLLLoss = CrossEntropyLoss，log 空间更稳定 |

</quiz>

## 第六题：CNN 代码分析

```python
class SimpleCNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.features = nn.Sequential(
            nn.Conv2d(3, 32, 3, padding=1), nn.ReLU(), nn.MaxPool2d(2),
            nn.Conv2d(32, 64, 3, padding=1), nn.ReLU(), nn.MaxPool2d(2),
            nn.Conv2d(64, 128, 3, padding=1), nn.ReLU(), nn.MaxPool2d(2),
        )
        self.classifier = nn.Sequential(
            nn.Flatten(),
            nn.Linear(128 * 4 * 4, 256),
            nn.ReLU(),
            nn.Dropout(0.5),
            nn.Linear(256, 5)
        )

    def forward(self, x):
        x = self.features(x)
        x = self.classifier(x)
        return x
```

<quiz>
(1) 特征提取部分有[[3]]层卷积，通道数变化：3→[[32]]→[[64]]→[[128]]

(2) 输入 32×32，128×4×4 计算过程：32→pool(16)→pool([[8]])→pool([[4]]) → 128×4×4

(3) 该模型使用的正则化：[[Dropout(0.5)]]

(4) 若输入改为 64×64，需修改：Linear(128×[[8]]×[[8]], 256) 或用 [[AdaptiveAvgPool2d]]

(5) 可学习参数层共[[5]]层（conv1, conv2, conv3, Linear(256), Linear(5)）

---

| 空位 | 答案 |
|------|------|
| (1) | 3 层卷积：$3 \to 32 \to 64 \to 128$ |
| (2) | 32 $\to$ pool(16) $\to$ pool(8) $\to$ pool(4) $\to$ $128\times4\times4$ |
| (3) | Dropout(0.5) 在第一个 FC 层之后 |
| (4) | Linear($128\times8\times8$, 256) 或 AdaptiveAvgPool2d |
| (5) | 5 层：conv1, conv2, conv3, Linear(256), Linear(5) |

</quiz>
