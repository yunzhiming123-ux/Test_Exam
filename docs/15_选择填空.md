## 一、选择题

<quiz>
第 1 题（神经网络基础）

以下关于神经元的描述，正确的是？

- [ ] A. 神经元计算为：输出 = 输入 × 权重，不需要偏置
- [ ] B. 激活函数的作用是降低模型复杂度
- [ ] C. 神经网络层数越深，参数量越少
- [x] D. 神经元先做加权求和加偏置，再经过激活函数得到输出

神经元计算：z = Σwᵢxᵢ + b → a = f(z)。偏置提供平移能力，激活函数引入非线性。
</quiz>

<quiz>
第 2 题（激活函数）

以下关于 ReLU 激活函数的描述，错误的是？

- [ ] A. 正区间导数为 1，缓解梯度消失
- [ ] B. 计算简单，只需 max(0, x)
- [x] C. 输出范围是 (-1, 1)，零均值
- [ ] D. 可能存在 Dead ReLU 问题

ReLU 输出范围是 [0, +∞)，不是 (-1, 1)。Tanh 输出才是 (-1, 1)。
</quiz>

<quiz>
第 3 题（损失函数）

分类任务中，`nn.CrossEntropyLoss()` 内部包含了什么操作？

- [ ] A. 仅计算交叉熵
- [x] B. LogSoftmax + NLLLoss
- [ ] C. Softmax + MSE
- [ ] D. Sigmoid + BCELoss

CrossEntropyLoss 先对 logits 做 LogSoftmax（log+softmax），再取正确类别的负对数（NLLLoss）。
</quiz>

<quiz>
第 4 题（优化器）

SGD + Momentum 相比普通 SGD 的主要优势是？

- [ ] A. 不需要设置学习率
- [x] B. 加入动量（惯性），加速收敛，减少震荡
- [ ] C. 每次更新使用更大的 batch
- [ ] D. 自动调整每个参数的学习率

SGD+Momentum 累积历史梯度方向作为动量，在梯度方向一致时加速，在震荡时平滑。Adam 才是自适应学习率。
</quiz>

<quiz>
第 5 题（CNN 卷积）

卷积操作中，padding="same" 的含义是？

- [ ] A. 输出通道数等于输入通道数
- [x] B. 输出特征图的空间尺寸等于输入尺寸（当 stride=1 时）
- [ ] C. 所有卷积核共享权重
- [ ] D. 不填充边缘

"Same padding" 通过补零使输出尺寸 = 输入尺寸 ÷ stride（stride=1 时相等）。计算公式：P = (K-1)/2。
</quiz>

<quiz>
第 6 题（CNN 池化）

关于池化层，以下描述正确的是？

- [ ] A. 池化层包含大量可训练参数
- [x] B. MaxPool2d(2,2) 将特征图尺寸减半
- [ ] C. 平均池化比最大池化更适合分类任务
- [ ] D. 池化层能增加特征图通道数

池化层无参数（纯数学操作）。MaxPool 保留最显著特征，是分类任务默认选择。池化只改变空间尺寸，不改变通道数。
</quiz>

<quiz>
第 7 题（RNN/LSTM）

LSTM 中遗忘门的激活函数为什么用 Sigmoid 而不是 Tanh？

- [ ] A. Sigmoid 计算更快
- [x] B. Sigmoid 输出 (0,1)，天然表示"门"的开闭程度
- [ ] C. Tanh 会导致梯度爆炸
- [ ] D. Sigmoid 输出和为 1

Sigmoid 输出范围 (0,1)，1=全保留，0=全遗忘。Tanh 输出 (-1,1) 无法表示"门"的通断语义。
</quiz>

<quiz>
第 8 题（Transformer）

Self-Attention 中为什么要除以 √d_k？

- [ ] A. 减小计算量
- [x] B. 防止点积过大导致 Softmax 梯度消失
- [ ] C. 增加模型复杂度
- [ ] D. 让注意力矩阵变成对称矩阵

Q·K^T 的方差随 d_k 线性增长，除以 √d_k 将方差稳定为 1，防止点积值过大进入 Softmax 饱和区（梯度接近 0）。
</quiz>

<quiz>
第 9 题（Dropout）

关于 Dropout，以下描述正确的是？

- [ ] A. Dropout 在训练和测试时都生效
- [ ] B. Dropout 增加了模型的可学习参数量
- [x] C. 训练时随机丢弃神经元，测试时全部参与
- [ ] D. Dropout 只用于卷积层

Dropout 训练时以概率 p 丢弃神经元（输出置 0），测试时所有神经元参与。PyTorch 自动处理训练/测试模式的差异。
</quiz>

<quiz>
第 10 题（数据维度）

PyTorch 中，一个 shape 为 `(32, 3, 224, 224)` 的张量，各维度的含义是？

- [ ] A. (通道, 批次, 高度, 宽度)
- [x] B. (批次, 通道, 高度, 宽度)
- [ ] C. (批次, 高度, 宽度, 通道)
- [ ] D. (通道, 高度, 宽度, 批次)

PyTorch 默认 NCHW 格式：batch=32, channel=3, height=224, width=224。
</quiz>

<quiz>
第 11 题（训练流程）

训练循环中 `optimizer.zero_grad()` 的正确位置是？

- [ ] A. 在 `loss.backward()` 之后
- [x] B. 在每次 forward 之前（或每个 batch 开始时）
- [ ] C. 在 `optimizer.step()` 之后
- [ ] D. 不需要写，PyTorch 自动清空

PyTorch 默认梯度会累积，必须在每个 batch 的 forward 之前清空梯度，否则梯度会累加。
</quiz>

<quiz>
第 12 题（评估指标）

以下哪种情况属于过拟合？

- [x] A. 训练集 loss 很低但验证集 loss 很高
- [ ] B. 训练集和验证集 loss 都很高
- [ ] C. 训练集 loss 缓慢下降，验证集也在下降
- [ ] D. 训练集 loss 出现 NaN

过拟合表现为训练集表现好（记住训练样本），验证集表现差（没有学到通用特征）。训练集和验证集 loss 都高是欠拟合。
</quiz>

第 14 题（前向/反向传播）

训练循环五步法，按顺序填写：

① [[optimizer.zero_grad()]]

② [[output = model(x)]]

③ [[loss = criterion(output, y)]]

⑤ [[optimizer.step()]]

---
① 清空梯度 ② 前向传播 ③ 计算损失 ④ 反向传播 ⑤ 更新参数

<quiz>
第 15 题（Epoch / Batch / Iteration）

Batch 的定义：[[一次喂给模型的样本数量]]

Iteration 的定义：[[完成一个batch的前向和反向，即一次参数更新]]

Epoch 的定义：[[所有训练数据被完整使用一遍]]

---
公式：Iterations per epoch = ⌈总样本数 / batch_size⌉
</quiz>

<quiz>
第 16 题（卷积公式）

卷积输出尺寸公式中：

H_out = [[⌊(H_in - K + 2P) / S⌋ + 1]]

其中 K = [[卷积核大小]]，P = [[填充圈数]]，S = [[步长]]

---
H_out = ⌊(H_in - K + 2P) / S⌋ + 1
</quiz>

<quiz>
第 17 题（参数量公式）

Linear 层参数量公式：[[D_in × D_out + D_out]]

Conv2D 层参数量公式：[[C_in × C_out × K² + C_out]]

---
Linear 含 bias：D_in × D_out + D_out<br>Conv2D 含 bias：C_in × C_out × K² + C_out
</quiz>

<quiz>
第 18 题（LSTM 核心公式）

LSTM 的三个门：[[遗忘门]]、[[输入门]]、[[输出门]]，全部用[[Sigmoid]]激活函数。

细胞状态更新：c_t = [[f_t ⊙ c_{t-1} + i_t ⊙ \tilde{c}_t]]

---
遗忘门(f_t)、输入门(i_t)、输出门(o_t)；全部用 Sigmoid<br>c_t = f_t ⊙ c_{t-1} + i_t ⊙ \tilde{c}_t
</quiz>

<quiz>
第 19 题（注意力公式）

Scaled Dot-Product Attention：

Attention(Q, K, V) = [[softmax(QK^T / √d_k) V]]

d_k = [[d_model]] / [[n_heads]]（Multi-Head Attention 中）

---
Attention(Q, K, V) = softmax(QK^T / √d_k) V<br>d_k = d_model / n_heads
</quiz>

<quiz>
第 20 题（归一化对比）

BatchNorm 归一化维度：[[batch维度（每个通道在batch上归一化）]]，常用于[[CNN]]

LayerNorm 归一化维度：[[feature维度（每个样本独立归一化）]]，常用于[[Transformer/RNN]]

---
BatchNorm → batch 维度 → CNN<br>LayerNorm → feature 维度 → Transformer / RNN
</quiz>

<quiz>
第 21 题（残差连接公式）

ResNet 残差块公式：H(x) = [[F(x) + x]]

反向传播时：∂H/∂x = [[∂F/∂x + 1]]

其中 [[+1 项]] 保证梯度不会消失。

---
H(x) = F(x) + x<br>∂H/∂x = ∂F/∂x + 1<br>"+1" 项为梯度提供了高速通道
</quiz>

<quiz>
第 22 题（梯度消失证明）

Sigmoid 导数公式：σ'(x) = [[σ(x)(1 - σ(x))]]

最大值为 [[0.25]]（在 x = [[0]] 时取得）

一个 N 层 Sigmoid 网络，梯度最多衰减到 [[0.25^N]]

---
σ'(x) = σ(x)(1 - σ(x)) ≤ 0.25
</quiz>

<quiz>
第 23 题（梯度裁剪）

梯度裁剪的代码：

`torch.nn.utils.[[clip_grad_norm_]]`(model.parameters(), max_norm=[[5.0]])

若所有参数梯度 L2 范数超过 max_norm，则新梯度 = [[原梯度 × (max_norm / ||grad||)]]

---
clip_grad_norm_，max_norm=5.0（常用值）<br>新梯度 = 原梯度 × (max_norm / ||grad||)
</quiz>

其中 V = [[词汇表大小（不同token的数量）]]，D = [[每个token的嵌入向量维度]]

参数量 = [[V × D]]

Embedding 参数矩阵 shape: (V, D)，参数量 = V × D
