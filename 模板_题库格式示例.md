> **本文档说明题库的推荐语法格式，以及如何让解析器正确识别不同题型。**

## 一、支持的题型

| 题型 | 识别标识 | 交互方式 |

## 二、推荐格式（新格式）

### 选择题

```markdown
**题目1**（选择题）：以下关于ReLU的描述，正确的是？
A. 输出范围在(0,1)之间
B. 可以缓解梯度消失问题
C. 在x<0时导数为0.01
D. 主要用于输出层

<details><summary>答案</summary>

**答案：B**

ReLU = max(0,x)，在正区间导数为1（不变），因此梯度可以无损传播，缓解了Sigmoid/Tanh的梯度消失问题。
A是Sigmoid特征，C是LeakyReLU特征，D错误（Softmax用于输出层多分类）。
</details>
```

### 填空题

```markdown
**题目2**（填空题）：写出交叉熵损失的公式：$$ L = \underline{\qquad\qquad} $$

<details><summary>答案</summary>

**答案：** $$ -\sum_{c} y_c \log(\hat{y}_c) $$

其中 $y_c$ 是真实标签（one-hot），$\hat{y}_c$ 是预测概率。
</details>
```

> 对于填空答案，系统会自动把 `|` 作为分隔符来匹配多个空。
> 例如答案写 `4 | 输入门 | 遗忘门 | 输出门` 表示四个填空的对应答案。

### 简答题

```markdown
**题目3**（简答题）：解释梯度消失的原因，并给出至少两种解决方案。

<details><summary>答案</summary>

**参考答案：**
梯度消失是指在深层网络中，反向传播时梯度逐层衰减，导致前面几层几乎不更新的现象。

**原因：**
- Sigmoid/Tanh的导数值小于1（Sigmoid导数最大0.25）
- 链式求导时多个小于1的数连乘→指数衰减

**解决方案：**
1. 使用ReLU激活函数（正区间导数为1）
2. 使用Batch Normalization
3. 使用残差连接（ResNet）
4. 使用LSTM/GRU替代SimpleRNN
</details>
```

### 代码分析题

```markdown
**题目4**（代码分析题）：分析以下代码，回答问题。

```python
class SimpleCNN(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()
        self.conv1 = nn.Conv2d(3, 32, 3, padding=1)  # 输入3通道(RGB)
        self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
        self.pool = nn.MaxPool2d(2, 2)
        self.fc = nn.Linear(64 * 8 * 8, num_classes)

    def forward(self, x):  # x: (B, 3, 32, 32)
        x = self.pool(F.relu(self.conv1(x)))  # → (B, 32, 16, 16)
        x = self.pool(F.relu(self.conv2(x)))  # → (B, 64, 8, 8)
        x = x.view(x.size(0), -1)
        return self.fc(x)
```

(1) 该网络输入图像尺寸是多少？
(2) 经过两次卷积+池化后，特征图尺寸是多少？
(3) `self.fc` 的输入维度为什么是 `64 * 8 * 8`？

<details><summary>答案</summary>

**参考答案：**
(1) 输入为 32×32 的RGB图像（3通道）
(2) 第一次: 32→16（MaxPool stride=2），第二次: 16→8，最终为 8×8×64通道
(3) 经过两次池化（每次减半），空间尺寸变为8×8，通道数64 → flatten后为 64×8×8=4096
</details>
```

## 三、兼容的旧格式

如果你已有采用旧格式的 .md 文件（如 `**Q1.** 问题...`），解析器也会自动识别：

```markdown
**Q1.** RNN的"循环"具体指什么？

<details><summary>答案</summary>

**A1.** 循环指 h_t 的计算依赖 h_{t-1}...

**A2.** 不变。W、U、b在所有时间步共享...
</details>
```

## 四、关键规则

1. **每道题的答案必须放在 `<details>` 块中**，解析器才能提取正确解答
2. **选择题选项**以 `A.`/`B.`/`C.`/`D.` 开头（中英文句号或顿号均可）
3. **填空题**中横线用 `___`（三个下划线）或 `\underline{}` LaTeX命令
4. **题型标记**写在题号的括号中：`（选择题）`、`（填空题）`、`（简答题）`、`（代码分析题）`
5. 即使不写题型标记，解析器也会根据内容自动推断题型
6. MathJax 公式支持 `$$ ... $$`（块级）和 `$ ... $`（行内）

## 五、快速上手

1. 复制上面的模板修改成你自己的题目
2. 将 .md 文件放在题库文件夹中
3. 打开 `exam_bank.py` 或 `深度学习题库.exe`
4. 左侧选择文件，顶部选择模式（学习/练习/考试）
5. 开始刷题！
