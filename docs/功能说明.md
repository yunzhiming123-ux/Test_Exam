# 功能说明

## 三种模式

### 学习模式
- 完整渲染 Markdown 内容（公式、代码高亮、表格）
- 每道题下方有「显示答案」按钮，点击展开 / 折叠
- 顶部搜索框：输入关键词即时过滤题目

### 练习模式
- 每道题独立作答，提交后立即判对错
- **选择题**：点击选项选中 → 点击「提交答案」
- **填空题**：在输入框中填写 → 自动记录 → 点击「提交答案」
- 顶部计分栏实时更新：已答 / 总题数 / 答对
- 正确答案高亮（绿色边框），错误选项划删除线

### 考试模式
- 所有题目统一计时
- 选项顺序随机打乱（防作弊）
- 点击「提交试卷」一次性提交所有答案
- 百分制评分 + 评语

## 题型识别

程序自动从 .md 文件中检测题型：

| 题型 | 判定依据 | 交互控件 |
|------|----------|----------|
| 选择题 | 含 A/B/C/D 选项 或 标记 `（选择题）` | 单选按钮 |
| 填空题 | 含 `___` 或 `\underline{}` | 文本输入框 |
| 简答题 | 无选项标记的普通问答 | 折叠答案 |
| 代码分析题 | 含代码块 | 折叠答案 |
| 计算题 | 标记 `（计算题）` | 折叠答案 |

## 快捷键

| 按键 | 功能 |
|------|------|
| `N` | 跳转到下一题 |

## 依赖

- Python 3.7+
- PyQt5 + PyQtWebEngine
- markdown
